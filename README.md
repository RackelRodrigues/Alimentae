<h1 align='center'> Alimentae</h1>

## ✨ Sobre o projeto

O **Alimentae** é uma landing page focada no segmento de alimentação e restaurantes, criada com o objetivo de estudar integrações modernas utilizando ferramentas e serviços do Google.

O projeto simula uma plataforma de busca de restaurantes, delivery e descoberta de estabelecimentos, utilizando uma interface moderna, responsiva e otimizada para experiência do usuário.

## 🎯 Objetivo

Este projeto foi desenvolvido principalmente para:

- Estudar APIs e serviços do Google
- Explorar funcionalidades de busca e localização
- Aprender integrações modernas no front-end
- Melhorar performance e experiência do usuário
- Criar interfaces responsivas e modernas

## 🚀 Funcionalidades

- Busca de restaurantes
- Sessão de restaurantes em destaque
- Estatísticas e informações visuais
- Interface responsiva
- Navegação fluida
- Design moderno

## 🔎 Busca de restaurantes

A landing page possui um campo de busca para restaurantes logo no início da aplicação.

Atualmente, essa funcionalidade possui foco visual e estatístico para estudos de interface e experiência do usuário.

Futuramente poderão ser adicionadas integrações reais utilizando APIs do Google.

## ⚙️ Instalação

Clone o projeto:

```bash
git clone https://github.com/seu-user/alimentae.git
```

Acesse a pasta:

```bash
cd alimentae
```

Instale as dependências:

```bash
npm install
```

Execute o projeto:

```bash
npm run dev
```
